
# Random Graph Generator							



## Overview
Simple random graph generator based on <a href ="http://www.cse.psu.edu/~kxm85/software/GTgraph/index.html">GTgraph</a>



## Installation ##
```python

python setup.py install
 
```


## Usage ##
- Just enter number of vertices
